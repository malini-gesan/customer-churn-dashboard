#📊 Customer Churn Analysis Dashboard

## 🔍 Project Overview

This project presents an interactive Power BI dashboard designed to analyze customer churn behavior and uncover key business insights.
The dashboard helps identify why customers are leaving and highlights patterns that can support better decision-making.

---

## 🎯 Objective

* To analyze customer churn trends
* To identify high-risk customer segments
* To provide actionable insights for improving customer retention

---

## 📌 Key Metrics

* **Total Customers:** 7,000+
* **Churn Rate:** 27%
* **Churned Customers:** 2,000+

---

## 📈 Dashboard Features

### 🔹 Customer Churn Distribution

* Visual representation of churn vs non-churn customers
* Helps understand overall churn percentage

### 🔹 Churn by Contract Type

* Comparison of churn across:

  * Month-to-month
  * One-year
  * Two-year contracts
* Identifies which contract type has the highest churn

### 🔹 Monthly Charges Analysis

* Customers grouped into bins based on monthly charges
* Shows how pricing impacts churn behavior

---

## 🧠 Key Insights

* Customers with **month-to-month contracts** have higher churn rates
* Higher **monthly charges** tend to increase churn probability
* Long-term contracts improve customer retention
